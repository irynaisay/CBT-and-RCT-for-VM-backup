# CBT and RCT for VM backup
## PROs and CONs

Backups is one of the many pain points for system administrators and their companies consequently. Companies develop and approve retention policies determining back up requirements, procedures, and options, which may vary depending on the needs and resources. Backups of virtual machines exposes even higher risks.

As [technopedia determines](https://www.techopedia.com/definition/4805/virtual-machine-vm), a virtual machine (VM) is a software or an operating system that runs applications and programs as well as performs multiple tasks as a real computer.

Changed Block Tracking (CBT) provides for incremental backups.
